# 更新记录

## v1.0.0 - 2026-04-20

这是 `Codex Session Bridge` 的第一个正式可试用版本。

### 本版范围

1. 已完成本地 `.codex` 会话发现、搜索、筛选、预览。
2. 已完成“恢复到 Codex”的主路径，可通过恢复包衔接上下文。
3. 已完成实验性 provider 同步、备份、加锁、日志、回滚基础设施。
4. 已支持标题补全增强，可优先使用 SQLite、索引、会话正文改善跨账号旧会话识别。
5. 当前版本仍以“恢复上下文”而非“恢复原生线程身份”为边界。

### 当前已验证结果

1. 可扫描本机 `%USERPROFILE%\\.codex` 下的真实会话。
2. 可将旧会话整理为恢复包并挂入当前 Codex 对话。
3. 切换账号后，Codex 可以基于恢复包继续承接上下文。
4. 实验性同步链路具备备份、日志、回滚保护。

### 当前已知边界

1. 恢复本质上仍是“把历史上下文恢复给新线程”，不是恢复原生 thread 身份。
2. 在部分场景下，Codex 仍需要当前窗口先打开项目目录才能正常发送消息。
3. 当前版本仍以本地试运行和工程验证为主。
